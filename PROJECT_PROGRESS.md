## 🔧 技术架构

### 后端
- **框架**: FastAPI (Python 3.x)
- **数据库**: MySQL
- **认证**: JWT Token
- **AI 服务**: 阿里云千问 API

### 前端
- **框架**: React 18 + TypeScript
- **UI 库**: Ant Design 5.x
- **图表**: ECharts
- **状态管理**: Zustand
- **构建工具**: Vite
